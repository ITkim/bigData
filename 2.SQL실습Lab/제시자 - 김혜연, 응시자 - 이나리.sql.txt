/**11.
입사번호가 7698보다 작거나 같은 사람들의 입사번호와 이름을 출력하라.
comment : 입사번호가 사원번호 인가요?

/**12.

comment 문제의 일관성을 위해 문제 10번과 같이 늦고, 빠르고 라는 형용사 대신 먼저 입사 하고, 나중에 입사한 이라고 고치면 좋을 것 같습니다.

/**13.
급여가 1600보다 크고[초과] 3000보다 작은[미만] 사람의 이름, 직업, 급여를 출력하라.

부서가 30번 부서이고 급여가 1500이상인 사람의 이름, 부서, 월급(sal)을 출력하라.
comment 이름, 부서, 월급중에서 부서명 혹은 부서번호를 구하라는 것인지 확실하게 해야할 것 같습니다. 여기서는 부서명이라고 가정하겠습니다.

/**22.

/**24.
