1. 모든 사원들의 평균 연봉을 출력하시오.

select avg(sal) from emp;

2. 부서별 사원수를 출력하시오.

select count(*) from emp group by deptno;

3. 직원수가 3명인 부서에 속한 사원번호와 직원이름을 출력하시오.

select ename,empno from emp where deptno in (select deptno from emp group by deptno having count(*)=3);

4. 상사의 직업이 MANAGER인 사원들의 연봉의 합을 출력하시오.

select sum(sal) from emp where job='MANAGER';

5. 'ACCOUNTING'부서에 속한 직원들의 모든 정보를 출력하시오.

select * from emp e,dept d where e.deptno=d.deptno and d.dname='ACCOUNTING';
